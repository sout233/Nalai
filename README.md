# Nalai
 
使用C#编写的下载器，支持异步多线程分块下载。

界面使用 [WPF UI](https://github.com/lepoco/wpfui) 实现 Fluent Design。

经测试，**Nalai**目前在Windows10以及Windows11上工作良好。

> [!IMPORTANT]
> 目前该项目正处于快速开发阶段，部分功能可能尚未实现。
